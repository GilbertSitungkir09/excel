1. Apa Itu Microsoft Excel?

Microsoft excel merupakan bagian dari Microsoft office yang berfungsi untuk pengolahan data dalam bentuk angka maupun perhitungan. Tampilan dari Microsoft excel ini berupa spreadsheet sehingga memudahkan penggunanya untuk mengoperasikannya.

Aplikasi Microsoft excel ini, merupakan pengolah data secara otomatis yang dapat berupa penghitungan dasar, penggunaan rumus-rumus, pemakaian fungsi, pembuatan grafik, pengolahan data dan table, serta manajemen data. Namun, dalam pemakaian fungsi anda harus paham betul arti dari fungsi tersebut serta logika excelnya. Karena jika salah dalam pemakaian fungsi, dapat menyebabkan kesalahan sewaktu membaca data.

2. Sejarah Microsoft Excel

Excel pertama kali hadir pada tahun 1985 dan dibuat oleh Doug Klunder. Fakta uniknya, Excel secara perdana dirilis tanpa memuat nama ‘Microsoft.’ Melainkan hanya Excel saja. Munculnya Excel ternyata mendapat banyak komplain dari berbagai perusahaan yang memiliki produk dengan nama serupa. Sesaat setelahnya, Microsoft langsung mengubahnya menjadi ‘Microsoft Excel.’

Pada tahun pertamanya, Excel hanya didistribusikan untuk Macintosh. Dua tahun setelahnya, tepatnya pada November 1987, Microsoft merilis Excel versi Windows 2.0. Pada tahun 1993, Excel dibekali bahasa pemrograman Visual Basic for Applications (VBA), yang menjadi tonggak fitur rumus otomatis dan akurat.

Rincian Versi Windows:

Excel 2 (1987) s/d Excel 5 (1993)
Excel 95 s/d Excel 2003
Excel 2007 s/d Office 2016

1. Math & Arithmetic (Matematika)

SUMMenjumlahkan seluruh angka dalam rentang sel.=SUM(A1:A10)
SUMIF / SUMIFSMenjumlahkan angka berdasarkan satu atau banyak kriteria.=SUMIF(A1:A10, ">50")
PRODUCTMengalikan seluruh angka dalam argumen.=PRODUCT(A1, B1)
SUBTOTALMenghasilkan subtotal dalam daftar (bisa mengabaikan sel tersembunyi).=SUBTOTAL(9, A1:A10)
ROUND / UP / DOWNMembulatkan angka ke digit tertentu (biasa/atas/bawah).=ROUND(10.556, 2)
INTMembulatkan angka ke bawah ke bilangan bulat terdekat.=INT(8.9)
ABSMenghasilkan nilai mutlak (positif) dari angka.=ABS(-5)
MODMenghasilkan sisa hasil bagi.=MOD(10, 3)
POWER / SQRTMenghitung pangkat atau akar kuadrat.=SQRT(16)
RAND / BETWEENMenghasilkan angka acak (biasa/dalam rentang).=RANDBETWEEN(1, 100)
CEILING / FLOORMembulatkan ke kelipatan terdekat (atas/bawah).=CEILING(2.5, 1)

2. Statistical (Statistik)

AVERAGE / IF / IFSMenghitung rata-rata angka (biasa/kriteria).=AVERAGE(A1:A5)
COUNT / COUNTA / BLANKMenghitung sel berisi angka, isi, atau kosong.=COUNT(A1:A10)
COUNTIF / COUNTIFSMenghitung jumlah sel berdasarkan kriteria.=COUNTIF(A1:A10, "Lulus")
MAX / MINMencari nilai tertinggi atau terendah.=MAX(A1:A10)
MEDIAN / MODEMencari nilai tengah atau nilai sering muncul.=MEDIAN(A1:A10)
LARGE / SMALLMencari nilai terbesar/terkecil ke-k.=LARGE(A1:A10, 2)
RANKMenentukan peringkat angka dalam daftar.=RANK(A1, A1:A10)
PERCENTILEMenghasilkan persentil ke-k.=PERCENTILE(A1:A10, 0.9)
STDEV / VARMenghitung standar deviasi atau varians.=STDEV(A1:A10)

3. Logical (Logika)

IF / IFSUji logika benar atau salah (tunggal/majemuk).=IF(A1>70, "Lulus", "Gagal")
AND / OR / NOTOperator logika penggabung kriteria.=AND(A1>0, B1>0)
IFERROR / IFNAMenangani pesan error pada rumus.=IFERROR(A1/B1, 0)
SWITCH / XOREvaluasi nilai atau logika eksklusif.=SWITCH(A1, 1, "Satu", 2, "Dua")

4. Lookup & Reference (Pencarian)

VLOOKUP / HLOOKUPMencari data vertikal atau horizontal.=VLOOKUP(A1, F1:G10, 2, 0)
XLOOKUPPencarian modern yang sangat fleksibel.=XLOOKUP(A1, F:F, G:G)
INDEX / MATCH / XMATCH

Kombinasi pencarian data tingkat lanjut.

=INDEX(G1:G10, MATCH(A1, F1:F10, 0))
OFFSET / INDIRECT / CHOOSE

Referensi sel dinamis atau pilihan indeks.

=CHOOSE(1, "A", "B")
ROW / COLUMN / ADDRESS

Informasi alamat baris atau kolom.

=ROW(A10)

5. Text Functions (Teks)

LEFT / RIGHT / MID / LEN

Manipulasi atau hitung karakter teks.

=LEFT("EXCEL", 2)
TRIM / UPPER / LOWER / PROPER

Format huruf dan bersihkan spasi.

=UPPER("halo")
CONCAT / TEXTJOIN / SUBSTITUTE

Gabung atau ganti teks.

=TEXTJOIN(", ", TRUE, A1, B1)
FIND / SEARCH / TEXT / VALUE

Cari posisi teks atau konversi format.

=TEXT(TODAY(), "DD/MM/YY")

6. Date & Time (Tanggal)

TODAY / NOW

Tanggal atau waktu saat ini.

=TODAY()
DAY / MONTH / YEAR / DATE

Ekstrak atau susun tanggal.

=YEAR(TODAY())
DATEDIF / NETWORKDAYS

Hitung selisih hari atau hari kerja.

=DATEDIF(A1, B1, "D")
EDATE / EOMONTH / WORKDAY

Manipulasi jatuh tempo tanggal.

=EOMONTH(TODAY(), 0)
