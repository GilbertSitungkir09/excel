1. Apa Itu Microsoft Excel?

Microsoft excel merupakan bagian dari Microsoft office yang berfungsi untuk pengolahan data dalam bentuk angka maupun perhitungan. Tampilan dari Microsoft excel ini berupa spreadsheet sehingga memudahkan penggunanya untuk mengoperasikannya.

Aplikasi Microsoft excel ini, merupakan pengolah data secara otomatis yang dapat berupa penghitungan dasar, penggunaan rumus-rumus, pemakaian fungsi, pembuatan grafik, pengolahan data dan table, serta manajemen data. Namun, dalam pemakaian fungsi anda harus paham betul arti dari fungsi tersebut serta logika excelnya. Karena jika salah dalam pemakaian fungsi, dapat menyebabkan kesalahan sewaktu membaca data.

2. Sejarah Microsoft Excel

Excel pertama kali hadir pada tahun 1985 dan dibuat oleh Doug Klunder. Fakta uniknya, Excel secara perdana dirilis tanpa memuat nama ‘Microsoft.’ Melainkan hanya Excel saja.

Sesaat setelahnya, Microsoft langsung mengubahnya menjadi ‘Microsoft Excel.’ Nama tersebut masih digunakan hingga sekarang. Pada tahun pertamanya, Excel hanya didistribusikan untuk Macintosh. Dua tahun setelahnya (1987), Microsoft merilis Excel untuk versi Windows 2.0.

Tepat pada 1993, Microsoft Excel sudah dibekali bahasa pemrograman Visual Basic for Applications (VBA). VBA-lah yang menjadi tonggak pertama Excel memiliki fitur rumus-rumus yang bisa digunakan secara otomatis dan akurat.

1. Math & Arithmetic (Matematika)

SUM Menjumlahkan seluruh angka dalam rentang sel. =SUM(A1:A10)
SUMIF / SUMIFS Menjumlahkan angka berdasarkan satu atau banyak kriteria. =SUMIF(B1:B10, ">100", C1:C10)
PRODUCT Mengalikan seluruh angka dalam argumen. =PRODUCT(A1, B1)
ROUND / UP / DOWN Membulatkan angka ke digit tertentu. =ROUND(12.345, 2)

2. Statistical (Statistik)

AVERAGE / IF / IFS Menghitung rata-rata angka. =AVERAGE(A1:A10)
COUNT / COUNTA / BLANK Menghitung jumlah sel terisi atau kosong. =COUNT(A1:A10)
MAX / MIN Mencari nilai tertinggi atau terendah. =MAX(A1:A10)

3. Logical (Logika)

IF / IFS Uji logika benar atau salah. =IF(A1>70, "Lulus", "Gagal")
IFERROR Menangani error pada rumus. =IFERROR(A1/B1, 0)

4. Lookup & Reference (Pencarian)

VLOOKUP / HLOOKUP Mencari data secara vertikal atau horizontal. =VLOOKUP(A2, F2:G10, 2, FALSE)
XLOOKUP Pencarian data modern dan fleksibel. =XLOOKUP(A2, F2:F10, G2:G10)

5. Text Functions (Teks)

LEFT / RIGHT / MID Mengambil karakter teks. =LEFT(A1, 5)
CONCAT / TEXTJOIN Menggabungkan teks. =CONCAT(A1, " ", B1)
