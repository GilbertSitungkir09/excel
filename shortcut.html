🛠️ Umum & Pengeditan Dasar

ALT + =
AutoSum: Menjumlahkan nilai sel di atas atau di sebelah kiri secara otomatis (Fungsi SUM cepat).
CTRL + C
Menyalin (Copy) sel atau rentang yang dipilih.
CTRL + V
Menempelkan (Paste) data yang telah disalin.
CTRL + Z
Membatalkan tindakan terakhir (Undo).
CTRL + Y
Mengulang tindakan yang dibatalkan (Redo).
CTRL + F
Membuka kotak dialog Find untuk mencari data.
CTRL + H
Membuka kotak dialog Replace (Cari dan Ganti).

🎨 Formatting (Pemformatan)

CTRL + 1
Membuka kotak dialog Format Cells.
CTRL + B
Menebalkan teks (Bold).
CTRL + I
Memiringkan teks (Italic).
ALT + ENTER
Membuat baris baru di dalam sel yang sama (Wrap Text manual).
CTRL + SHIFT + $
Menerapkan format mata uang (Currency) dengan 2 desimal.
CTRL + SHIFT + %
Menerapkan format persentase tanpa desimal.

🧭 Navigasi & Manipulasi Data

CTRL + Arah Panah
Lompat langsung ke ujung baris atau kolom data terakhir.
F4
Mengunci sel (Absolute Reference / menambah simbol $ pada rumus).
CTRL + SHIFT + L
Mengaktifkan atau menonaktifkan fitur AutoFilter pada tabel.
CTRL + ;
Memasukkan Tanggal hari ini secara statis.
CTRL + SHIFT + :
Memasukkan Waktu saat ini secara statis.
CTRL + PG UP / PG DN
Berpindah antar tab Worksheet (Sheet) di bagian bawah.
