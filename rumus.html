1. Math & Arithmetic (Matematika)

Pemula SUM Menjumlahkan seluruh angka dalam rentang sel terpilih.
=SUM(A1:A10)
Menengah SUMIF / SUMIFS Menjumlahkan angka berdasarkan satu atau banyak kriteria.
=SUMIF(A1:A10, ">50", B1:B10)
PRODUCT Mengalikan semua angka yang diberikan sebagai argumen.
=PRODUCT(A1, B1)
SUBTOTAL Menghasilkan subtotal dalam daftar atau database.
=SUBTOTAL(9, A1:A10)
ROUND / UP / DOWN Membulatkan angka ke jumlah digit yang ditentukan.
=ROUND(A1, 2)
INT Membulatkan angka ke bawah ke bilangan bulat terdekat.
=INT(8.9)
ABS Menghasilkan nilai mutlak (positif) dari suatu angka.
=ABS(-10)
MOD Menghasilkan sisa setelah angka dibagi pembagi.
=MOD(10, 3)
POWER Menghasilkan hasil angka yang dipangkatkan.
=POWER(5, 2)
SQRT Menghasilkan akar kuadrat positif dari suatu angka.
=SQRT(16)
RAND / BETWEEN Menghasilkan angka acak (di antara rentang tertentu).
=RANDBETWEEN(1, 100)
CEILING / FLOOR Membulatkan angka ke kelipatan terdekat (atas/bawah).
=CEILING(2.5, 1)
📊

2. Statistical (Statistik)

Pemula AVERAGE / IF / IFS Menghitung rata-rata aritmatika dari argumen.
=AVERAGE(A1:A10)
Pemula COUNT / COUNTA / BLANK Menghitung jumlah sel yang berisi angka atau data.
=COUNTA(A1:A10)
Menengah COUNTIF / COUNTIFS Menghitung jumlah sel berdasarkan kriteria tertentu.
=COUNTIF(B:B, "Laki-laki")
Pemula MAX / MIN Mencari nilai terbesar atau terkecil dalam rentang.
=MAX(A1:A10)
MEDIAN / MODE Mencari nilai tengah atau nilai yang sering muncul.
=MEDIAN(A1:A10)
LARGE / SMALL Mencari nilai terbesar atau terkecil ke-n.
=LARGE(A1:A10, 2)
RANK Menghasilkan peringkat angka dalam daftar angka.
=RANK(A1, A1:A10)
PERCENTILE Menghasilkan persentil ke-k dari nilai dalam rentang.
=PERCENTILE(A1:A10, 0.9)
STDEV / VAR Menghitung standar deviasi atau varians sampel.
=STDEV(A1:A10)
⚖️

3. Logical (Logika)

Pemula IF / IFS Uji logika untuk menghasilkan satu nilai jika benar, lainnya jika salah.
=IF(A1>70, "Lulus", "Tidak")
AND / OR / NOT Operator logika untuk menggabungkan beberapa kriteria.
=AND(A1>0, B1>0)
Menengah IFERROR / IFNA Menghasilkan nilai kustom jika rumus menghasilkan error.
=IFERROR(A1/B1, 0)
SWITCH Mengevaluasi satu nilai terhadap daftar nilai.
=SWITCH(A1, 1, "Senin", 2, "Selasa")
XOR Logika OR Eksklusif.
=XOR(A1>0, B1>0)
🔍

4. Lookup & Reference (Pencarian Data)

Menengah VLOOKUP / HLOOKUP Mencari nilai di kolom pertama tabel secara vertikal/horizontal.
=VLOOKUP(A2, $F$2:$G$10, 2, FALSE)
Menengah XLOOKUP Versi modern VLOOKUP yang lebih fleksibel dan kuat.
=XLOOKUP(A2, F2:F10, G2:G10)
LOOKUP Mencari nilai baik dalam satu baris atau satu kolom.
=LOOKUP(A1, F:F, G:G)
Menengah INDEX / MATCH / XMATCH Kombinasi pencarian data paling fleksibel.
=INDEX(G1:G10, MATCH(A1, F1:F10, 0))
OFFSET / CHOOSE Mengembalikan referensi sel atau memilih dari daftar.
=CHOOSE(2, "A", "B", "C")
INDIRECT / ADDRESS Mengubah teks menjadi referensi sel asli.
=INDIRECT("A1")
ROW / COLUMN Mengembalikan nomor baris atau kolom suatu referensi.
=ROW(A1)
🔡

5. Text Functions (Teks)

LEFT / RIGHT / MID Mengambil karakter teks dari kiri, kanan, atau tengah.
=LEFT(A1, 5)
LEN / TRIM Menghitung panjang teks atau menghapus spasi berlebih.
=TRIM(A1)
UPPER / LOWER / PROPER Mengubah format huruf besar, kecil, atau kapital kata.
=PROPER(A1)
CONCAT / TEXTJOIN Menggabungkan beberapa sel teks menjadi satu.
=TEXTJOIN(", ", TRUE, A1:A5)
TEXT / VALUE Konversi angka ke format teks atau sebaliknya.
=TEXT(TODAY(), "dd/mm/yyyy")
FIND / SEARCH Mencari posisi karakter tertentu dalam teks.
=FIND("@", A1)
REPLACE / SUBSTITUTE Mengganti bagian teks tertentu dengan teks baru.
=SUBSTITUTE(A1, "Lama", "Baru")
📅

6. Date & Time (Tanggal & Waktu)

TODAY / NOW Menampilkan tanggal hari ini atau waktu saat ini.
=TODAY()
DATE / TIME Membuat data format tanggal atau waktu.
=DATE(2026, 5, 3)
DAY / MONTH / YEAR Ekstrak angka hari, bulan, atau tahun.
=YEAR(TODAY())
HOUR / MINUTE / SECOND Ekstrak angka jam, menit, atau detik.
=HOUR(NOW())
DATEDIF Menghitung selisih antara dua tanggal (Hari/Bulan/Tahun).
=DATEDIF(A1, B1, "Y")
EDATE / EOMONTH Mengambil tanggal jatuh tempo atau akhir bulan.
=EOMONTH(TODAY(), 0)
WORKDAY / NETWORKDAYS Menghitung hari kerja efektif di antara tanggal.
=NETWORKDAYS(A1, B1)
💰

7. Financial (Keuangan)

PMT / FV / PV / RATE Menghitung pembayaran cicilan, nilai masa depan, atau bunga.
=PMT(0.1/12, 60, 100000000)
NPV / IRR / XIRR / XNPV Analisis arus kas investasi (Net Present Value).
=IRR(A1:A5)
SLN / DB Menghitung penyusutan aset secara garis lurus/menurun.
=SLN(100, 10, 5)
⚡

8. Dynamic Array (Excel Modern)

Mahir FILTER / UNIQUE / SORT Menyaring, mengurutkan, atau mengambil data unik otomatis.
=UNIQUE(A1:A100)
SORTBY / SEQUENCE Mengurutkan array atau membuat daftar angka urut.
=SEQUENCE(10)
RANDARRAY Membuat array angka acak secara dinamis.
=RANDARRAY(5, 5)
TAKE / DROP Mengambil atau membuang baris/kolom dari array.
=TAKE(A1:C10, 5)
CHOOSECOLS / TOCOL / TOROW Re-organisasi kolom menjadi satu kolom atau baris.
=TOCOL(A1:C10)
🗄️

9. Database Functions

DSUM / DAVERAGE Menjumlahkan atau merata-rata field di database sesuai kriteria.
=DSUM(A1:C100, "Harga", E1:E2)
DCOUNT / DMAX / DMIN Statistik database berdasarkan kriteria tabel terpisah.
=DMAX(A1:C100, "Penjualan", E1:E2)
ℹ️

10. Information Functions

ISBLANK / ISNUMBER / ISTEXT Pengecekan apakah sel kosong, berisi angka, atau teks.
=ISNUMBER(A1)
ISERROR / ISNA Pengecekan apakah sel mengandung pesan error.
=ISERROR(A1)
TYPE / CELL Mendapatkan informasi detail tentang properti sel.
=CELL("format", A1)
⚙️

11. Engineering

CONVERT Konversi angka dari satu sistem pengukuran ke sistem lain.
=CONVERT(1, "m", "cm")
DEC2BIN / BIN2DEC Konversi bilangan desimal ke biner atau sebaliknya.
=BIN2DEC("1010")
COMPLEX Membuat bilangan kompleks dari koefisien real dan imajiner.
=COMPLEX(3, 4)
🌐

12. Web / Advanced Data

WEBSERVICE Mengambil data dari layanan web langsung ke Excel.
=WEBSERVICE("http://api.com")
FILTERXML Mengambil data spesifik dari XML menggunakan XPath.
=FILTERXML(A1, "//data")
