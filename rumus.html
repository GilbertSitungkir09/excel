1. Math & Arithmetic

SUMMenjumlahkan angka.
=SUM(A1:A10)
SUMIF / SUMIFSJumlahkan angka dengan kriteria.
=SUMIF(A1:A10, ">50")
PRODUCTMengalikan angka.
=PRODUCT(A1:A5)
SUBTOTALSubtotal dalam daftar.
=SUBTOTAL(9, A1:A10)
ROUND / UP / DOWNPembulatan angka.
=ROUND(A1, 2)
INTBilangan bulat terdekat.
=INT(8.9)
ABSNilai mutlak (positif).
=ABS(-5)
MODSisa hasil bagi.
=MOD(10, 3)
POWERPangkat angka.
=POWER(5, 2)
SQRTAkar kuadrat.
=SQRT(16)
RAND / BETWEENAngka acak.
=RANDBETWEEN(1, 100)
CEILING / FLOORPembulatan kelipatan.
=CEILING(2.5, 1)

2. Statistical

AVERAGE / IF / IFSRata-rata.
=AVERAGE(A1:A10)
COUNT / COUNTA / BLANKHitung jumlah sel.
=COUNTA(A1:A10)
COUNTIF / COUNTIFSHitung dengan kriteria.
=COUNTIF(A1:A10, "Apple")
MAX / MINNilai tertinggi/terendah.
=MAX(A1:A10)
MEDIAN / MODENilai tengah/sering muncul.
=MEDIAN(A1:A10)
LARGE / SMALLNilai ke-n.
=LARGE(A1:A10, 2)
RANKPeringkat angka.
=RANK(A1, A1:A10)
PERCENTILENilai persentil.
=PERCENTILE(A1:A10, 0.9)
STDEV / VARStandar deviasi.
=STDEV(A1:A10)
⚖️

3. Logical

IF / IFSUji logika.
=IF(A1>70, "Lulus", "Gagal")
AND / OR / NOT / XORGabungan kriteria logika.
=AND(A1>0, B1>0)
IFERROR / IFNACegah error sel.
=IFERROR(A1/B1, 0)
SWITCHEvaluasi terhadap daftar.
=SWITCH(A1, 1, "Senin")
