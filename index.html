1. Apa Itu Microsoft Excel?

Microsoft excel merupakan bagian dari Microsoft office yang berfungsi untuk pengolahan data dalam bentuk angka maupun perhitungan. Tampilan dari Microsoft excel ini berupa spreadsheet sehingga memudahkan penggunanya untuk mengoperasikannya.

Aplikasi Microsoft excel ini, merupakan pengolah data secara otomatis yang dapat berupa penghitungan dasar, penggunaan rumus-rumus, pemakaian fungsi, pembuatan grafik, pengolahan data dan table, serta manajemen data. Namun, dalam pemakaian fungsi anda harus paham betul arti dari fungsi tersebut serta logika excelnya. Karena jika salah dalam pemakaian fungsi, dapat menyebabkan kesalahan sewaktu membaca data.

2. Sejarah Microsoft Excel

Excel pertama kali hadir pada tahun 1985 dan dibuat oleh Doug Klunder. Fakta uniknya, Excel secara perdana dirilis tanpa memuat nama ‘Microsoft.’ Melainkan hanya Excel saja. Munculnya Excel ternyata mendapat banyak komplain dari berbagai perusahaan yang memiliki produk dengan nama serupa. Sesaat setelahnya, Microsoft langsung mengubahnya menjadi ‘Microsoft Excel.’ Nama tersebut masih digunakan hingga sekarang.

Pada tahun pertamanya, Excel hanya didistribusikan untuk Macintosh. Dua tahun setelahnya, tepatnya pada November 1987, Microsoft merilis Excel versi Windows 2.0. Pada tahun 1993, Excel dibekali bahasa pemrograman Visual Basic for Applications (VBA), yang menjadi tonggak fitur rumus otomatis dan akurat.

3. Mengenal Tampilan & Istilah Dasar

RibbonKumpulan tab menu (Home, Insert, dll) yang berisi tool atau perintah.
ToolbarKumpulan tombol untuk akses cepat ke perintah tertentu.
WorksheetLembar kerja tunggal di dalam Excel yang terdiri dari sel.
WorkbookFile Excel yang dapat menampung satu atau lebih Worksheet.
RowsBaris horisontal yang diwakili oleh angka (1, 2, 3...).
ColumnsKolom vertikal yang diwakili oleh huruf (A, B, C...).
CellPertemuan antara baris dan kolom (Contoh: A1).
Formula BarTempat untuk melihat, mengetik, atau mengedit data/rumus di sel aktif.
Sheet TabsTab di bagian bawah untuk berpindah antar lembar kerja.

Istilah Dasar:

RangeSekumpulan sel yang dipilih secara bersamaan (Contoh: A1:B10).
FormulaPersamaan matematis untuk menghitung nilai (Diawali tanda =).
FunctionRumus siap pakai dari Excel (Contoh: SUM, AVERAGE).
AutofillFitur untuk mengisi data secara otomatis berdasarkan pola.
FormattingMengatur tampilan data (Warna, Huruf, Border).

1. Math & Arithmetic (Matematika)

SUMMenjumlahkan seluruh angka dalam rentang sel.=SUM(A1:A10)
SUMIF / SUMIFSMenjumlahkan angka berdasarkan satu atau banyak kriteria.=SUMIF(A1:A10, ">50")
PRODUCTMengalikan seluruh angka dalam argumen.=PRODUCT(A1, B1)
SUBTOTALMenghasilkan subtotal dalam daftar (bisa mengabaikan sel tersembunyi).=SUBTOTAL(9, A1:A10)
ROUND / UP / DOWNMembulatkan angka ke digit tertentu (biasa/atas/bawah).=ROUND(10.556, 2)
INTMembulatkan angka ke bawah ke bilangan bulat terdekat.=INT(8.9)
ABSMenghasilkan nilai mutlak (positif) dari angka.=ABS(-5)
MODMenghasilkan sisa hasil bagi.=MOD(10, 3)
POWERMenghasilkan hasil angka yang dipangkatkan.=POWER(5, 2)
SQRTMenghasilkan akar kuadrat dari suatu angka.=SQRT(16)
RAND / BETWEENMenghasilkan angka acak (biasa/dalam rentang).=RANDBETWEEN(1, 100)
CEILING / FLOORMembulatkan ke kelipatan terdekat (atas/bawah).=CEILING(2.5, 1)

2. Statistical (Statistik)

AVERAGE / IF / IFSMenghitung rata-rata angka (biasa/kriteria).=AVERAGE(A1:A5)
COUNT / COUNTA / BLANKMenghitung sel berisi angka, isi, atau kosong.=COUNT(A1:A10)
COUNTIF / COUNTIFSMenghitung jumlah sel berdasarkan kriteria.=COUNTIF(A1:A10, "Lulus")
MAX / MINMencari nilai tertinggi atau terendah.=MAX(A1:A10)
MEDIAN / MODEMencari nilai tengah atau nilai sering muncul.=MEDIAN(A1:A10)
LARGE / SMALLMencari nilai terbesar/terkecil ke-k.=LARGE(A1:A10, 2)
RANKMenentukan peringkat angka dalam daftar.=RANK(A1, A1:A10)
PERCENTILEMenghasilkan persentil ke-k.=PERCENTILE(A1:A10, 0.9)
STDEV / VARMenghitung standar deviasi atau varians.=STDEV(A1:A10)

3. Logical (Logika)

IF / IFSUji logika benar atau salah (tunggal/majemuk).=IF(A1>70, "Lulus", "Gagal")
AND / OR / NOTOperator logika penggabung kriteria.=AND(A1>0, B1>0)
IFERROR / IFNAMenangani pesan error pada rumus.=IFERROR(A1/B1, 0)
SWITCHMengevaluasi ekspresi terhadap daftar nilai dan hasilkan kecocokan pertama.=SWITCH(A1, 1, "Satu", 2, "Dua")
XORLogika eksklusif (bernilai benar jika jumlah argumen benar adalah ganjil).=XOR(A1>0, B1>0)

4. Lookup & Reference (Pencarian)

VLOOKUP / HLOOKUPMencari data vertikal atau horizontal.=VLOOKUP(A1, F1:G10, 2, 0)
XLOOKUPPencarian modern yang sangat fleksibel.=XLOOKUP(A1, F:F, G:G)
LOOKUPMencari nilai baik dalam satu baris atau satu kolom.=LOOKUP(A1, F1:F10, G1:G10)
INDEX / MATCH / XMATCHKombinasi pencarian data tingkat lanjut.=INDEX(G1:G10, MATCH(A1, F1:F10, 0))
OFFSET / INDIRECT / CHOOSEReferensi sel dinamis atau pilihan indeks.=CHOOSE(1, "A", "B")
ADDRESSMembuat alamat sel sebagai teks berdasarkan nomor baris dan kolom.=ADDRESS(1, 1)
ROW / COLUMNInformasi alamat baris atau kolom.=ROW(A10)

5. Text Functions (Teks)

LEFT / RIGHT / MID / LENManipulasi atau hitung karakter teks.=LEFT("EXCEL", 2)
TRIM / UPPER / LOWER / PROPERFormat huruf dan bersihkan spasi.=UPPER("halo")
CONCAT / CONCATENATE / TEXTJOINMenggabungkan teks (TEXTJOIN bisa pakai pembatas).=TEXTJOIN(", ", TRUE, A1, B1)
SUBSTITUTE / REPLACEMengganti teks lama dengan baru.=SUBSTITUTE(A1, "Lama", "Baru")
FIND / SEARCH / TEXT / VALUECari posisi teks atau konversi format.=TEXT(TODAY(), "DD/MM/YY")

6. Date & Time (Tanggal)

TODAY / NOWTanggal atau waktu saat ini.=TODAY()
DAY / MONTH / YEAR / DATEEkstrak atau susun tanggal.=YEAR(TODAY())
HOUR / MINUTE / SECOND / TIMEEkstrak atau susun waktu.=HOUR(NOW())
DATEDIF / NETWORKDAYSHitung selisih hari atau hari kerja.=DATEDIF(A1, B1, "D")
EDATE / EOMONTH / WORKDAYManipulasi jatuh tempo tanggal.=EOMONTH(TODAY(), 0)

7. Financial (Keuangan)

PMT / FV / PV / RATEHitung cicilan, nilai masa depan, atau bunga.=PMT(0.1/12, 12, -1000000)
NPV / IRR / XIRR / XNPVAnalisis investasi dan arus kas.=NPV(0.1, A1:A5)
SLN / DBHitung penyusutan aset tetap.=SLN(100, 10, 5)

8. Dynamic Array (Modern)

FILTER / UNIQUE / SORT / SORTBYOtomatisasi daftar data dinamis.=UNIQUE(A1:A10)
SEQUENCE / RANDARRAYBuat urutan atau array acak.=SEQUENCE(5)
TAKE / DROP / CHOOSECOLS / TOCOL / TOROWReorganisasi struktur array.=TOCOL(A1:B2)

9. Database Functions

DSUM / DCOUNT / DAVERAGEFungsi statistik pada kriteria database.=DSUM(A1:C10, "Skor", E1:E2)
DMAX / DMINNilai tertinggi/terendah di database.=DMAX(A1:C10, "Nilai", E1:E2)

10. Information Functions

ISBLANK / ISNUMBER / ISTEXTPengecekan tipe isi sel.=ISNUMBER(A1)
ISERROR / ISNA / TYPE / CELLInformasi status atau properti sel.=ISERROR(A1)

11. Engineering

CONVERT / DEC2BIN / BIN2DECKonversi satuan dan sistem bilangan.=CONVERT(1, "m", "cm")
COMPLEXMembuat bilangan kompleks.=COMPLEX(3, 4)

12. Web / Advanced

WEBSERVICE / FILTERXMLPengambilan data dari web atau XML.=WEBSERVICE("http://api.com")
