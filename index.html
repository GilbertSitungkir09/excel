1. Apa Itu Microsoft Excel?

Microsoft excel merupakan bagian dari Microsoft office yang berfungsi untuk pengolahan data dalam bentuk angka maupun perhitungan. Tampilan dari Microsoft excel ini berupa spreadsheet sehingga memudahkan penggunanya untuk mengoperasikannya.

Aplikasi Microsoft excel ini, merupakan pengolah data secara otomatis yang dapat berupa penghitungan dasar, penggunaan rumus-rumus, pemakaian fungsi, pembuatan grafik, pengolahan data dan table, serta manajemen data. Namun, dalam pemakaian fungsi anda harus paham betul arti dari fungsi tersebut serta logika excelnya. Karena jika salah dalam pemakaian fungsi, dapat menyebabkan kesalahan sewaktu membaca data.

2. Sejarah Microsoft Excel

Excel pertama kali hadir pada tahun 1985 dan dibuat oleh Doug Klunder. Fakta uniknya, Excel secara perdana dirilis tanpa memuat nama ‘Microsoft.’ Melainkan hanya Excel saja. Munculnya Excel ternyata mendapat banyak komplain dari berbagai perusahaan yang memiliki produk dengan nama serupa. Sesaat setelahnya, Microsoft langsung mengubahnya menjadi ‘Microsoft Excel.’

Pada tahun pertamanya, Excel hanya didistribusikan untuk Macintosh. Dua tahun setelahnya, tepatnya pada November 1987, Microsoft merilis Excel untuk versi Windows 2.0. Keunggulan Excel mulai mendominasi pasar setelah tahun 1993, di mana Excel dibekali bahasa pemrograman Visual Basic for Applications (VBA) yang memungkinkan fitur rumus otomatis dan akurat.

3. Mengapa Belajar Excel Penting?

  • Skill Wajib Dunia Kerja: Menjadi standar kompetensi administrasi di hampir semua bidang.
  • Keuangan Pribadi & Bisnis: Sangat berguna untuk manajemen laba rugi hingga budgeting rumah tangga.
  • Meningkatkan Produktivitas: Mengolah ribuan data dalam hitungan detik dengan otomatisasi.

1. Math & Arithmetic

SUM

Menjumlahkan seluruh angka dalam rentang sel.

SUMIF / SUMIFS

Menjumlahkan angka berdasarkan satu atau banyak kriteria.

PRODUCT

Mengalikan seluruh angka dalam argumen.

SUBTOTAL

Menghitung subtotal dalam daftar (bisa mengabaikan sel tersembunyi).

ROUND / UP / DOWN

Membulatkan angka ke jumlah digit tertentu.

INT

Membulatkan angka ke bawah ke bilangan bulat terdekat.

ABS

Menghasilkan nilai mutlak (positif) dari angka.

MOD

Menghasilkan sisa hasil bagi.

POWER / SQRT

Menghitung pangkat atau akar kuadrat.

RAND / BETWEEN

Menghasilkan angka acak.

CEILING / FLOOR

Membulatkan ke kelipatan terdekat.

2. Statistical

AVERAGE / IF / IFS

Menghitung rata-rata.

COUNT / COUNTA / BLANK

Menghitung jumlah sel (angka/isi/kosong).

COUNTIF / COUNTIFS

Menghitung sel berdasarkan kriteria.

MAX / MIN / MEDIAN / MODE

Mencari nilai tertinggi, terendah, tengah, atau sering muncul.

LARGE / SMALL / RANK

Mencari urutan nilai terbesar/terkecil atau peringkat.

PERCENTILE / STDEV / VAR

Perhitungan statistik lanjut.

3. Logical

IF / IFS

Uji logika tunggal atau bertingkat.

AND / OR / NOT / XOR

Operator logika penggabung kriteria.

IFERROR / IFNA

Menangani pesan error pada rumus.

SWITCH

Mengevaluasi nilai terhadap daftar pasangan.

4. Lookup & Reference

VLOOKUP / HLOOKUP / XLOOKUP

Mencari data secara vertikal, horizontal, atau fleksibel.

INDEX / MATCH / XMATCH

Kombinasi pencarian data tingkat lanjut.

OFFSET / INDIRECT / CHOOSE

Referensi sel dinamis.

ROW / COLUMN / ADDRESS

Informasi lokasi sel.

5. Text Functions

Manipulasi karakter teks.

TRIM / UPPER / LOWER / PROPER

CONCAT / TEXTJOIN / SUBSTITUTE

FIND / SEARCH / TEXT / VALUE
