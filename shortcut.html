🛠️ Umum & Pengeditan Dasar

CTRL + O
Membuka buku kerja (Open workbook).
CTRL + S
Menyimpan buku kerja (Save workbook).
CTRL + W
Menutup buku kerja (Close workbook).
ALT + =
AutoSum: Menjumlahkan nilai sel di atas atau di sebelah kiri secara otomatis.
CTRL + C
Menyalin (Copy) sel atau rentang yang dipilih.
CTRL + X
Memotong (Cut) bagian yang dipilih / Memindahkan sel.
CTRL + V
Menempelkan (Paste) data yang telah disalin.
CTRL + ALT + V
Membuka kotak dialog Paste Special (Tempel Spesial).
CTRL + Z
Membatalkan tindakan terakhir (Undo).
CTRL + Y
Mengulang tindakan yang dibatalkan (Redo).
DELETE
Menghapus konten sel.
SHIFT + F10
Membuka menu konteks (Klik kanan). Sama dengan menekan tombol Windows Menu.
CTRL + F
Membuka kotak dialog Find untuk mencari data.
CTRL + H
Membuka kotak dialog Replace (Cari dan Ganti).
F2
Mengedit sel aktif dan meletakkan kursor di akhir konten sel.
SHIFT + F2
Menyisipkan atau mengedit catatan sel (Cell Note).
CTRL + SHIFT + F2
Menyisipkan atau membalas komentar berulir (Threaded Comment).
F7
Pemeriksaan ejaan (Spelling) pada lembar kerja.

📑 Navigasi Tab (Ribbon)

ALT atau F10
Mengaktifkan tombol akses cepat pada Ribbon.
TAB atau SHIFT + TAB
Memindahkan fokus ke perintah-perintah pada Ribbon.
Panah Kiri / Kanan
Pindah antar item di dalam Ribbon atau antar menu.
SPASI atau ENTER
Mengaktifkan tombol/menu yang sedang dipilih di Ribbon.
CTRL + SHIFT + F10
Menampilkan Tooltip (Info alat) untuk elemen Ribbon yang fokus.
CTRL + F1
Meluaskan (Expand) atau menciutkan (Collapse) Ribbon.
ALT , H
Pergi ke tab Beranda (Home).
ALT , N
Pergi ke tab Sisipkan (Insert).
ALT , P
Pergi ke tab Tata Letak Halaman (Page Layout).
ALT , M
Pergi ke tab Rumus (Formula).
ALT , A
Pergi ke tab Data.
ALT , W
Pergi ke tab Tampilan (View).

🎨 Formatting (Pemformatan)

CTRL + 1
Membuka kotak dialog Format Cells.
CTRL + SHIFT + F
Format Font di dialog Format Cells (Atau CTRL+SHIFT+P).
CTRL + B atau 2
Menebalkan teks (Bold).
CTRL + I atau 3
Memiringkan teks (Italic).
CTRL + U atau 4
Menggarisbawahi teks (Underline).
CTRL + 5
Menerapkan atau menghapus coretan (Strikethrough).
ALT , H , H
Memilih warna isian (Fill color).
ALT , H , A , C
Meratakan isi sel ke tengah (Center align).
ALT , H , B
Menambahkan batas garis luar dari tab beranda (Add borders).
CTRL + SHIFT + &
Menerapkan batas garis luar langsung (Outline border).
CTRL + SHIFT + _
Menghapus batas garis luar sel.
ALT + ENTER
Membuat baris baru di dalam sel yang sama (Wrap Text manual).
CTRL + SHIFT + ~
Menerapkan format angka Umum (General).
CTRL + SHIFT + $
Menerapkan format Mata Uang (Currency) 2 desimal.
CTRL + SHIFT + %
Menerapkan format Persentase tanpa desimal.
CTRL + SHIFT + ^
Menerapkan format Ilmiah (Scientific) 2 desimal.
CTRL + SHIFT + #
Menerapkan format Tanggal (Hari, Bulan, Tahun).
CTRL + SHIFT + @
Menerapkan format Waktu (Jam, Menit, AM/PM).
CTRL + SHIFT + !
Menerapkan format Angka (Pemisah ribuan, 2 desimal).

🧭 Navigasi & Manipulasi Data

Panah Atas / Bawah / Kiri / Kanan
Pindah satu sel ke arah yang sesuai.
TAB / SHIFT + TAB
Pindah satu sel ke kanan / Pindah ke sel sebelumnya.
CTRL + Arah Panah
Lompat langsung ke ujung baris atau kolom data terakhir.
END Lalu Arah Panah
Masuk mode End, pindah ke sel berikutnya yang tidak kosong.
CTRL + END
Pindah ke sel terakhir yang digunakan di Worksheet.
CTRL + SHIFT + END
Memperluas seleksi sel sampai sel terakhir yang digunakan.
CTRL + HOME
Pindah ke awal worksheet (Ke sel A1).
HOME + SCROLL LOCK
Pindah ke sel sudut kiri atas jendela saat Scroll lock aktif.
PAGE UP / PAGE DOWN
Pindah satu layar penuh ke atas atau ke bawah.
ALT + PAGE UP / DOWN
Pindah satu layar ke kiri atau ke kanan.
CTRL + PG UP / PG DN
Berpindah ke sheet sebelumnya / berikutnya dalam buku kerja.
CTRL + SHIFT + Scroll Mouse
Scroll horizontal (Atas untuk ke kiri, Bawah untuk ke kanan).
CTRL + ALT + = / -
Melakukan Zoom in (=) atau Zoom out (-).
F4
Mengunci sel (Absolute Reference / menambah simbol $ pada rumus).
ALT + Panah Bawah
Buka daftar pilihan Validasi Data di sel (Atau menu Drop-down).
CTRL + SHIFT + L
Mengaktifkan/menonaktifkan AutoFilter pada tabel.
CTRL + SHIFT + +
Membuka dialog Insert untuk menyisipkan sel/baris/kolom.
CTRL + -
Membuka dialog Delete untuk menghapus sel/baris/kolom.
CTRL + 9
Menyembunyikan baris yang dipilih (Hide rows).
CTRL + 0
Menyembunyikan kolom yang dipilih (Hide columns).
ALT , H , D , C
Menghapus kolom (Delete column).
CTRL + D
Gunakan perintah Fill Down (Menyalin konten sel teratas ke sel di bawahnya).
CTRL + ;
Memasukkan Tanggal hari ini secara statis.
CTRL + SHIFT + :
Memasukkan Waktu saat ini secara statis.
CTRL + `
Beralih antara menampilkan nilai sel atau menampilkan rumusnya.
CTRL + '
Menyalin rumus dari sel di atasnya persis.
CTRL + 8
Menampilkan atau menyembunyikan simbol outline.
CTRL + 6
Beralih sembunyikan/tampilkan objek (seperti gambar/chart).
CTRL + ALT + 5
Masuk navigasi bentuk mengambang (lalu tekan Tab berulang). Tekan ESC untuk keluar.
CTRL + K
Membuka dialog Sisipkan Hyperlink.
CTRL + Q
Menampilkan opsi Analisis Cepat (Quick Analysis).
CTRL + L atau T
Membuka dialog Buat Tabel (Create Table).
CTRL + SHIFT + G
Membuka dialog Statistik Buku Kerja (Workbook Statistics).
