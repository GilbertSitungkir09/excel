Apa Itu Microsoft Excel?

Microsoft excel merupakan bagian dari Microsoft office yang berfungsi untuk pengolahan data dalam bentuk angka maupun perhitungan. Tampilan dari Microsoft excel ini berupa spreadsheet sehingga memudahkan penggunanya untuk mengoperasikannya.

Aplikasi Microsoft excel ini, merupakan pengolah data secara otomatis yang dapat berupa penghitungan dasar, penggunaan rumus-rumus, pemakaian fungsi, pembuatan grafik, pengolahan data dan table, serta manajemen data. Namun, dalam pemakaian fungsi anda harus paham betul arti dari fungsi tersebut serta logika excelnya.

1. Math & Arithmetic (Matematika)

SUM

Menjumlahkan seluruh angka dalam rentang sel.

SUMIF / SUMIFS

Menjumlahkan angka berdasarkan satu atau banyak kriteria tertentu.

PRODUCT

Mengalikan seluruh angka yang ada di dalam argumen.

SUBTOTAL

Menghasilkan subtotal dalam daftar atau database (bisa mengabaikan sel tersembunyi).

ROUND / UP / DOWN

Membulatkan angka ke jumlah digit tertentu (biasa, ke atas, atau ke bawah).

INT

Membulatkan angka ke bawah ke bilangan bulat terdekat.

ABS

Menghasilkan nilai mutlak (positif) dari suatu angka.

MOD

Menghasilkan sisa hasil bagi dari dua angka.

POWER

Menghasilkan hasil angka yang dipangkatkan.

SQRT

Menghasilkan akar kuadrat dari suatu angka.

RAND / BETWEEN

Menghasilkan angka acak (biasa atau di antara rentang tertentu).

CEILING / FLOOR

Membulatkan angka ke atas atau ke bawah ke kelipatan terdekat.

2. Statistical (Statistik)

AVERAGE / IF / IFS

Menghitung rata-rata (biasa atau berdasarkan kriteria tertentu).

COUNT / COUNTA / BLANK

Menghitung sel berisi angka, sel tidak kosong, atau sel yang kosong.

COUNTIF / COUNTIFS

Menghitung jumlah sel berdasarkan satu atau banyak kriteria.

MAX / MIN

Mencari nilai tertinggi atau terendah dalam suatu rentang.

MEDIAN / MODE

Mencari nilai tengah atau nilai yang paling sering muncul.

LARGE / SMALL

Mencari nilai terbesar ke-k atau terkecil ke-k.

RANK

Menentukan peringkat suatu angka dalam daftar.

PERCENTILE

Menghasilkan persentil ke-k dari nilai dalam rentang.

STDEV / VAR

Menghitung standar deviasi atau varians (simpangan data).

3. Logical (Logika)

IF / IFS

Melakukan uji logika tunggal atau bertumpuk.

AND / OR / NOT

Uji apakah semua syarat terpenuhi, salah satu terpenuhi, atau membalikkan hasil logika.

IFERROR / IFNA

Menangani pesan error (#VALUE!, #N/A) dengan teks kustom.

SWITCH

Mengevaluasi satu ekspresi terhadap daftar nilai dan hasilkan hasil pertama yang cocok.

XOR

Logika eksklusif (bernilai benar jika jumlah argumen yang benar adalah ganjil).

4. Lookup & Reference (Pencarian Data)

VLOOKUP / HLOOKUP

Mencari data secara vertikal (kolom) atau horizontal (baris).

XLOOKUP

Fungsi pencarian modern yang lebih kuat dan fleksibel dari VLOOKUP.

INDEX / MATCH / XMATCH

Kombinasi pencarian data yang paling fleksibel melampaui kemampuan VLOOKUP.

CHOOSE

Memilih satu nilai dari daftar berdasarkan nomor indeks.

OFFSET / INDIRECT

Mengacu pada rentang secara dinamis atau berdasarkan teks alamat sel.

ROW / COLUMN

Menampilkan nomor baris atau nomor kolom dari suatu sel.

5. Text Functions (Teks)

LEFT / RIGHT / MID

Mengambil karakter dari kiri, kanan, atau tengah teks.

LEN / TRIM

Menghitung jumlah karakter atau menghapus spasi berlebih.

UPPER / LOWER / PROPER

Mengubah teks jadi huruf besar, kecil, atau huruf besar di awal kata.

CONCAT / TEXTJOIN

Menggabungkan teks (TEXTJOIN bisa menggunakan pembatas otomatis).

FIND / SEARCH

Mencari posisi karakter tertentu dalam teks.

SUBSTITUTE / REPLACE

Mengganti teks lama dengan teks baru.

6. Date & Time (Tanggal & Waktu)

TODAY / NOW

Menampilkan tanggal hari ini atau waktu saat ini (real-time).

DATE / TIME

Membuat format tanggal atau waktu dari angka terpisah.

DAY / MONTH / YEAR

Mengambil angka hari, bulan, atau tahun dari sebuah tanggal.

DATEDIF

Menghitung selisih antara dua tanggal (dalam hari, bulan, atau tahun).

NETWORKDAYS

Menghitung jumlah hari kerja (senin-jumat) di antara dua tanggal.

7. Financial (Keuangan)

PMT

Menghitung pembayaran angsuran pinjaman secara berkala.

FV / PV

Menghitung nilai masa depan (Future Value) atau nilai saat ini (Present Value).

NPV / IRR / XIRR

Analisis investasi (Net Present Value dan Internal Rate of Return).

SLN / DB

Menghitung penyusutan aset (garis lurus atau saldo menurun).

8. Dynamic Array (Excel Modern)

FILTER

Menampilkan data yang memenuhi kriteria secara dinamis.

UNIQUE

Mengambil daftar nilai yang unik (tanpa duplikat) dari suatu rentang.

SORT / SORTBY

Mengurutkan data secara otomatis.

SEQUENCE

Membuat daftar angka berurutan secara otomatis.

TAKE / DROP

Mengambil atau menghapus baris/kolom tertentu dari array.

9. Database Functions

DSUM / DAVERAGE

Menghitung total atau rata-rata pada database berdasarkan kriteria tabel.

DMAX / DMIN

Mencari nilai tertinggi/terendah dalam database.

10. Information Functions

ISBLANK / ISNUMBER / ISTEXT

Memeriksa apakah sel kosong, berisi angka, atau teks (menghasilkan TRUE/FALSE).

ISERROR / ISNA

Memeriksa apakah sel mengandung error.

CELL

Mendapatkan informasi tentang format, lokasi, atau isi sel.

11. Engineering

CONVERT

Mengonversi angka dari satu sistem pengukuran ke sistem lain.

DEC2BIN / BIN2DEC

Mengonversi bilangan Desimal ke Biner atau sebaliknya.

12. Web / Advanced Data

WEBSERVICE

Mengambil data dari layanan web di internet.

FILTERXML

Mengambil data spesifik dari XML menggunakan XPath.
